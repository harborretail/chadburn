Cdma

Cdma: any

Dictionary of available signal information for the CDMA1x access technology. This dictionary is composed of a string key, with an associated data which contains type-specific information.

"rssi" The CDMA1x RSSI (Received Signal Strength Indication), in dBm, given as a floating point value (signature "d").

"ecio" The CDMA1x Ec/Io, in dBm, given as a floating point value (signature "d").

"error-rate" Frame error rate, in percentage value, given as a floating point value (signature "d").
